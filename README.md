# .txt merger
A simple one page project to upload, rearrange and merge your .txt files.
### Includes
- one HTML file with a simple JS script (not a lot of error checking, however)
- style.css (I wish I used tailwind instead, but maybe that would've been an overkill for something so small)
### Flow
1. Press the upload button
2. Select files
3. Drag and rearrange them
4. Enter custom filename
5. Merge & download!
### Why this?
I wanted to learn a little bit of JavaScript, since I mostly use other languages, so this seemed small yet complicated enough to understand the JS syntax and re-test my CSS skills, too. Also, a while ago, I needed to get some transctipts into one file and felt like it's quite a shame the website I used to merge them didn't have the option to name or rearrange the files, so I made my own!
### How to use
Either go to https://tekvickaaa.github.io/txt-merger/
or clone the repo and open the HTML
